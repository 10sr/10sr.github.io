Explore filesystem of stopped docker container

Feb 5, 2020

停止したコンテナにあったファイルをみたい。

以下のようにするとできる。

# コンテナの ID を探す（先頭のハッシュ）
docker ps -a | grep NAME

# 新しくイメージを作成する
docker commit CONTAINER_ID t

# 実行
docker run -it t bash -i
